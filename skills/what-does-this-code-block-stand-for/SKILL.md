## $code_block 에 대해 처리해야할 내용 목록.

1. **함수, 변수의 줄임말 설명**: 예를들어 C언어 함수 getc 의 c 는 character 를 의미한다라는 식의 부연설명 주석으로 추가.

## 출력 형식

축약된 문자를 대문자로 강조하고 실제 의미와 함께 보기 쉽게 출력.

```c
getc() -> get Charcter : 한 문자를 입력받습니다.
```
